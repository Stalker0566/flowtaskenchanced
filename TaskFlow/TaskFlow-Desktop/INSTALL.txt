🛠️ COMO INSTALAR:

1. ABRA O PROMPT DE COMANDO (CMD):
   - Pressione Windows + R
   - Digite: cmd
   - Pressione Enter

2. NAVEGUE ATÉ A PASTA:
   cd "C:\Users\andko\Desktop\TaskFlow\TaskFlow-Desktop"

3. INSTALE AS DEPENDÊNCIAS:
   npm install

4. EXECUTE EM MODO DESENVOLVIMENTO:
   npm run dev

5. PARA CRIAR O INSTALADOR:
   npm run build
   npm run dist
