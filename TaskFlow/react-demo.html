TaskFlow + React + PHP + MySQL

📋 React TodoList Component

What you're seeing:

This is a fully functional React component that communicates with your PHP backend and MySQL database. The component handles:

  • ✅ Adding new tasks
  • ✅ Marking tasks as complete/incomplete
  • ✅ Deleting individual tasks
  • ✅ Clearing all tasks
  • ✅ Real-time updates via AJAX
  • ✅ Error handling and loading states

🛠️ Technical Implementation

Project Structure:

TaskFlow/ ├── react/ # React components │ ├── src/ │ │ ├── components/ │ │ │ └── TodoList.jsx # React TodoList component │ │ └── main.jsx # React entry point │ ├── package.json # React dependencies │ └── vite.config.js # Vite build configuration ├── php/ │ └── tasks-api.php # PHP API for tasks ├── db/ │ └── schema.sql # Database schema (includes tasks table) └── js/ └── react-build/ # Built React components (after build)

How it works:

  1. React Component: TodoList.jsx renders the UI and manages state
  2. AJAX Communication: Uses axios to communicate with PHP API
  3. PHP API: tasks-api.php handles CRUD operations
  4. MySQL Database: Tasks are stored in the 'tasks' table
  5. Real-time Updates: UI updates immediately after API calls

🚀 Setup Instructions

Prerequisites:

Make sure you have Node.js and npm installed on your system.

Step 1: Install Dependencies

cd react npm install

Step 2: Build React Components

npm run build

This will create the built files in js/react-build/

Step 3: Update Database

Run the updated schema.sql to create the tasks table:

-- Import the updated schema.sql in phpMyAdmin -- This will create the 'tasks' table for React integration

Step 4: Include React in Your Pages

Add this to any HTML page where you want React components:

<!-- Include the built React components --> <script type="module" src="js/react-build/main.js"></script> <!-- Container for React component --> <div id="react-todo-container"></div>

🔧 Development Mode

For Development:

cd react npm run dev

This starts Vite dev server on http://localhost:3000

API Endpoints:

  • GET /api/tasks.php - Get all tasks
  • POST /api/tasks.php - Add/update/delete tasks
